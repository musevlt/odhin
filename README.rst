Installation
============

::

    git clone https://git-cral.univ-lyon1.fr/raphael.bacher/deblend

With setup.py
^^^^^^^^^^^^^

::

    python setup.py install

With conda
^^^^^^^^^^

::

    conda env create -f environment.yml
    conda activate odhin3
